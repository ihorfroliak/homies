# Architecture Decision Records (ADR)

ADR — це короткі записи важливих архітектурних рішень: **контекст** (які сили діяли), **рішення** (що обрали), **наслідки** (плюси й компроміси) та **розглянуті альтернативи**. Кожен ADR незмінний: якщо рішення змінюється, додається новий ADR, який позначає попередній як `Superseded`.

> Мова цих ADR — українська (робоча мова проєкту). Переклад англійською для міжнародного портфоліо — окреме завдання (Чат 14).
